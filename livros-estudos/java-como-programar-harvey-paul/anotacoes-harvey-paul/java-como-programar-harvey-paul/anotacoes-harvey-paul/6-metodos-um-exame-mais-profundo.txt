# 6.7 Promoção e coerção de argumentos
* Converter um número double em número int fará com que a parte fracioanda se perca, ou seja, ocorrendo uma perca de dados nos decimais.
* O valor dos tipos mais baixos devem ser sempre convertidos para os tipos maiores, não ocorrendo ao contrário, se não tenderá a perca de dados.
* Uma promoção em Java é basicamente converter um tipo de dado em um tipo maior na hierarquia

# 6.9 Estudo de caso: geração segura de números aleatórios
* A classe 'Random' gera números aleatórios que não são seguros
* A classe 'SecureRandom' gera números aleatórios que são seguros
